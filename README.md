# Damage Indicators

Acidglow's Damage Indicators is a NeoForge mod that displays floating damage numbers when entities take damage.

## Features

- Shows floating damage numbers above damaged living entities.
- Categorizes damage as low, high, critical, or entity/environment damage.
- Lets players toggle indicators, adjust text size, and customize category colors.
- Includes an in-game config screen, opened by default with `P`.

## Requirements

- Minecraft 1.21.11
- NeoForge 21.11.42 or newer for Minecraft 1.21.11
- Java 21

## Building

```sh
./gradlew build
```

The built mod jar is generated in `build/libs/`.
